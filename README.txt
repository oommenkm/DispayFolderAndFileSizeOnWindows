TITLE
    DispayFolderAndFileSizeOnWindows
    
DESCRIPTION
    This simple code will display the size of the folders and files in a given
    folder on a windows system.
    
    The windows system yet to show the folder size in the windows explorer.

DEPENDENCIES
    This applications requires Python Installed on you system.

PLATFORMS    
    Linux:
        Not Applicable
    
    Windows:
        Works Well
    
USGAE
    python DispayFolderAndFileSizeOnWindows.py
